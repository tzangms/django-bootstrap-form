Twitter Bootstrap for Django Form.

A simple Django template tag to work with `Bootstrap <http://twitter.github.com/bootstrap/>`_


Usage
======

Use it like this, simple.

.. code-block::

   {{ form|bootstrap }}

or

.. code-block::
	{{ form.field_name|bootstrap }}
